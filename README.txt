Программа: Отель
Основные сущности:
    Hotel (hotelId, hotelName, address)
    Room (hotelId, roomId)
    Guest (guestId, guestName)
    Booking (id, hotelId, roomId, guestId, startDate, endDate)
Программа умеет создавать, просматривать, изменять и удалять каждую из сущностей.
    Также реализована проверка пересечения дат, а также показ свободных номеров на период,
    занятости отеля в выбранную дату (количество номеров), всех броней для
    конкретного гостя, а также списка занятых номеров на определённую дату
    и расписание отеля на период.
В программе доступна регистрация пользователей и авторизация во время выполнения запросов.
    Пользователи с ролью USER могут использовать методы GET, POST, PUT
    Пользователи с ролью ADMIN могут использовать методы GET, POST, PUT, DELETE.
